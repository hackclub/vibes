Why Vibes?

Ok I get it, this is controversial. This YSWS program was not started because I (Jared) or Hack Club loves AI and wants to fast track a Terminator-like timeline. It is possible to use AI as a learning tool. This program is attempting to do this! Learning is hard. Learning to code is even harder. This is because they not only need to learn a programming language (even if it is basic like HTML + CSS), they have to learn all the other aspects of making a website go live, like everything with Git. All of this can be completely foreign to a beginner. I’ve been in clubs where they’ve never heard of GitHub and making a profile for the first time, with no idea why they are making one. Then they are told to make a repository, add index and css files, along with other completely new concepts. It can become overwhelming and sometimes discouraging. This program attempts to allow everyone to focus on just the basics while experimenting a little bit with some established code.

What is Vibes?

This program is broken into three parts. By the end, you will have a live website built by you with the assistance of good vibes (aka ChatGPT).

Wait, we can use AI!?

Yes! AI can be a controvertial topic, but it's good learning tool when used appropriately. In this program beginners can build complete pages while learning the basics of web development.

How it works

🧠 Part One

Prompt an AI to code a website about the topic of your choice.

🎨 Part Two

Edit the code yourself to change styling and content.

🚀 Part Three

Add new features to improve your website and make it your own.

How to participate

Join a club, you need to be part of a club to particpate in with this event, head over to the club map to find one near you! Then can complete everything independently through your email. Each part will be sent to you - just follow the instructions in each form.

The Reward 🍕

Once your club finishes, you'll celebrate together with pizza (or another snack chosen by your leader!).
